# BaiduMapDemo
用百度地图实现了  关键字搜索   定位      反地理编码搜索   移动地图获取地图中间的经纬度   

 仿qq发送位置 简单版
 
 编译环境:
 
 Android Studio 版本 3.5
 
 Gradle 3.4.2
 
 minSdkVersion = 21
 
 targetSdkVersion = 23
 
 compileSdkVersion = 28
 
 supportVersion = '28.0.0'
